# AI Chat Widget — AIチャット実装

> サイト内にAIチャットウィジェットを組み込む。

## 起動条件

- 「AIチャット付けて」「チャットボット」「アバター」

## Steps

1. **チャットUI設計**: ドロワー型 / モーダル型 / インライン型
2. **AI接続**: Gemini / OpenAI / Claude API
3. **アバター設計**: 静止画 / アニメーション
4. **実装**: フローティングボタン + チャットパネル
5. **UXテスト**: レスポンシブ + キーボードナビゲーション

## Do Not

- チャットボタンがコンテンツを隠す（右下固定 + 適切なマージン）
- 自動ポップアップ（ユーザーのクリックで開く）
