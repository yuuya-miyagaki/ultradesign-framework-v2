# Craft Portfolio — ポートフォリオ制作

> 作品が主役になるポートフォリオサイトを制作する。

## 起動条件

- 「ポートフォリオ作って」「作品集」「自己紹介サイト」

## Steps

1. **ポートフォリオタイプ決定**: デザイナー / エンジニア / 写真家 / アーティスト
2. **レイアウト選定**: グリッド / マソンリー / フルスクリーン / タイムライン
3. **作品表示方式**: ライトボックス / 詳細ページ / インラインプレビュー
4. **実装**: モバイルファーストCSS + スムーズなトランジション
5. **レビュー**: AIスロップチェック + レスポンシブ確認

## Design Principles

- 作品が主役。UIは脇役。
- ナビゲーションは最小限。
- 余白を贅沢に使う。
- ホバーエフェクトで探索を促す。

## Output

- ポートフォリオ HTML/CSS/JS 実装

## Do Not

- UIを過度に装飾して作品を埋没させる
- 全作品を一度に表示（ページネーション or 遅延読み込み推奨）
