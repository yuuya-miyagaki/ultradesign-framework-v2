# Craft Product Site — プロダクトサイト制作

> SaaS・プロダクトの価値を伝え、ユーザー獲得を最大化するサイトを制作する。

## 起動条件

- 「プロダクトサイト作って」「SaaSサイト」「製品ページ」

## Steps

1. **プロダクト理解**: 何を解決するか、差別化ポイント、ターゲット
2. **セクション構成**: ヒーロー → 課題提示 → ソリューション → 機能 → 料金 → CTA
3. **機能紹介方式**: Bento Grid / タブ切替 / スクロール連動
4. **料金表設計**: ターゲットに最適化（定番テンプレ禁止）
5. **実装 & レビュー**

## Key Sections

- **ヒーロー**: 一文で価値を伝える + CTA
- **Social Proof**: ロゴ / 数値 / テスティモニアル
- **Feature Grid**: Bento Grid 推奨
- **Pricing**: ターゲット価値観に合わせた独自構成

## Do Not

- Basic/Pro/Enterprise の定番料金表テンプレをそのまま使う
- 機能リストの羅列（ベネフィットで語る）
