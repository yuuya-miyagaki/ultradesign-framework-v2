# Design System — デザインシステム構築

> プロジェクト全体で一貫したビジュアル言語を定義する。

## 起動条件

- 「デザインシステム作って」「テーマを定義」「カラー提案」

## Steps

1. **カラーパレット定義**: `data/colors.csv` 参照、OKLCH色空間、60-30-10ルール
2. **タイポグラフィ定義**: `data/typography.csv` 参照、Variable Fonts優先
3. **スペーシングスケール定義**: 4px基準のスケール
4. **コンポーネントルール定義**: ボタン、カード、入力フィールド等の統一ルール
5. **CSS変数テーマファイル生成**

## Output

- デザインシステムスペック in `docs/specs/`
- CSS変数テーマファイル

## Do Not

- 個別コンポーネントのルール未定義でUI制作を開始
