# Craft Ecommerce — ECサイトUI制作

> コンバージョンを最大化する、プレミアムなECサイトUIを制作する。

## 起動条件

- 「ECサイト作って」「ショップ」「商品ページ」「オンラインストア」

## Steps

1. **ショップタイプ決定**: アパレル / デジタル / 食品 / ハンドメイド
2. **ページ構成**: トップ → カテゴリ → 商品一覧 → 商品詳細 → カート → チェックアウト
3. **商品一覧レイアウト**: グリッド / リスト / カルーセル
4. **商品詳細設計**: イメージギャラリー / スペック / レビュー / 関連商品
5. **カート・チェックアウト**: ミニカート / ステップフォーム / ゲスト購入

## Design Principles

- **商品写真が主役**: 大きく・美しく・高品質に
- **購入ファンネルを短縮**: 最小クリック数で購入完了
- **信頼性の表示**: レビュー、安全バッジ、返品ポリシー
- **モバイル購入体験**: スマホでのスムーズなチェックアウト

## Key Components

- 商品カード（ホバーで2枚目画像表示）
- サイズ/カラーセレクター
- ミニカート（ドロワー型）
- フィルター&ソート

## Do Not

- 商品を小さく表示する
- チェックアウトステップを隠す
- 会員登録を強制する（ゲスト購入を提供）
