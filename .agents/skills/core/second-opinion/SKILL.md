# Second Opinion — セカンドオピニオン

> デザイン決定や技術選択について、独立した視点でのレビューを提供する。

## 起動条件

- 「セカンドオピニオンお願い」「別の意見を聞きたい」
- 繰り返し失敗した後

## Steps

1. **コンテキスト整理**: 現在の状況、選択肢、制約
2. **独立評価**: 既存の決定に縛られず、ゼロベースで評価
3. **代替案提示**: 現在の方針と異なるアプローチを提案
4. **比較マトリクス**: メリット/デメリットの比較表

## Output

- セカンドオピニオンレポート

## Do Not

- 既存の決定を無条件に肯定
- 実行可能性を無視した理想論
